# Mobile App Assets

This directory contains the assets for the Echo Breaker mobile app.

## Required Assets

You need to create the following image assets:

### Icon (icon.png)
- Size: 1024x1024 px
- Format: PNG
- No transparency
- Content: Echo Breaker logo

### Adaptive Icon (adaptive-icon.png)
- Size: 1024x1024 px
- Format: PNG
- For Android
- Safe zone: 432x432 px centered
- Content: Echo Breaker logo

### Splash Screen (splash.png)
- Size: 1284x2778 px (or larger)
- Format: PNG
- Background: #0f172a (dark blue)
- Content: Echo Breaker logo centered

### Favicon (favicon.png)
- Size: 48x48 px
- Format: PNG
- Content: Echo Breaker logo icon

## Temporary Placeholders

Until you create the actual assets, you can use placeholder images.
The app will still build and run, but the icons will be generic.

## Design Guidelines

- Use the ApexSalesAI / ProofLayer branding
- Primary color: Indigo (#6366f1)
- Dark theme colors: #0f172a, #1e293b
- Keep the design clean and professional
